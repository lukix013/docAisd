📄 Plik nagłówkowy: prz_wiel.h

Plik prz_wiel.h zawiera deklaracje funkcji i definicję struktury Point, służące do operacji geometrycznych związanych z przynależnością punktu do wielokąta.

Definicje

  • struct Point – struktura reprezentująca punkt na płaszczyźnie kartezjańskiej, z polami x i y typu int.

Funkcje

  • bool czy_wspoliniowe(Point a, Point b, Point c ) – sprawdza, czy trzy punkty leżą na jednej prostej.
  • bool do_krawedzi(Point p, Point a, Point b ) – sprawdza, czy punkt p leży na odcinku wyznaczonym przez punkty a i b.
  • bool przecina_polprosta (Point p, Point a, Point b ) – określa, czy półprosta pozioma wychodząca z punktu p przecina odcinek [a, b].
  • bool punkt_w_wielokacie (Point p, const std::vector &wielokat ) – zwraca true, jeśli punkt p znajduje się wewnątrz lub na krawędzi wielokąta reprezentowanego przez wektor punktów wielokat.

Uwagi

  • Plik jest zabezpieczony przed wielokrotnym dołączaniem poprzez dyrektywy #ifndef, #define i #endif.
  • Wszystkie funkcje operują na punktach całkowitoliczbowych.
