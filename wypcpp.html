Funkcje i zmienne związane z wyznaczaniem otoczki wypukłej

Globalna zmienna:

  • Point p0 – punkt odniesienia, najniższy i najbardziej na lewo punkt z zestawu.

Funkcje:

int det(Point p1, Point p2, Point p3)
Oblicza wartość wyznacznika (iloczynu wektorowego) dla trzech punktów, co określa orientację kątową (skret lewo-/prawoskręt lub współliniowość).
int distanceSq(Point p1, Point p2)
Oblicza kwadrat odległości euklidesowej między punktami p1 i p2. Wykorzystywane do porównywania odległości przy sortowaniu punktów współliniowych.
bool comparePoints(Point p1, Point p2)
Funkcja porównująca punkty do sortowania względem kąta względem punktu p0. Zwraca true, jeśli p1 powinien być przed p2 w porządku sortowania.
int findFirst(const std::vector<Point>& points)
Znajduje indeks punktu o najmniejszej współrzędnej y, a w przypadku remisu najmniejszej x. To będzie punkt p0 dla dalszych obliczeń.
std::vector<Point> convexHull(std::vector<Point> points)
Główna funkcja zwracająca otoczkę wypukłą dla danego zestawu punktów metodą Graham Scan. Zwraca wektor punktów tworzących otoczkę wypukłą.

Opis działania funkcji convexHull:

  1. Jeśli liczba punktów jest mniejsza niż 3, zwraca pusty wektor (otoczka nie istnieje).
  2. Wybiera punkt p0 jako punkt startowy (najniższy lewy).
  3. Sortuje pozostałe punkty względem kąta od p0.
  4. Tworzy początkową otoczkę z trzech pierwszych punktów.
  5. Iteruje po pozostałych punktach, dodając je do otoczki i usuwając punkty powodujące prawoskręty (lub współliniowość), aby zachować właściwość otoczki wypukłej.
