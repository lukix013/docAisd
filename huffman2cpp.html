📄 Plik: huffman2.cpp

Plik zawiera pełną implementację klasy Huff,amEncoder, odpowiadającej za kompresję i dekompresję plików przy uzyciu algorytmu Huffmana

Główne funkcjonalności klasy HuffmanEncoder

  • Konstruktor: Inicjalizacja kodera na podstawie nazwy pliku.
  • decompressFile(const std::string& inputFilePath,const std::string& outputFilePath) – kopiuje z pliku do pliku (plik zakodowany -> odkodowany), ponieważ jest to funkcja statyczna nie musimy inicjalizować obiektu, używamy: HuffmanEncoder::decompressFile(copressedFile, uncompressedFile).
  • compressToFile(const std::string& outputFile) – kopiuje z pliku do pliku (plik nie kodowany -> plik zakodowany)
  • buildTree() - buduje drzewo Huffmana
  • printCodesRecursive(Node* node, const std::string& prefix) - wypisuje zakodowane wartości dla każdego znaku
  • destroyTree(Node* node) - usuwa drzewo z pamięci
  • generateCodes(Node* node, const std::string& prefix) - tworzy kody binarne dla każdego znaku

Struktury pomocnicze

  • Node – reprezentuje węzeł drzewa Huffmana. Zawiera char chr, int data, Node* left, right, dostępna tylko w obrembie klasy HuffmanEncoder

Uwagi techniczne

  • Zakładany format pliku: najpierw zakodowane drzewo, następnie dane – ważne przy dekompresji.
  • Plik wyjściowy zostanie nadpisany jeśli już istnieje.

Zależności

  • #include <string>, <map>, <queue>, <fstream>, <iostream>, <vector>
